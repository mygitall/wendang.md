## 目录结构

```
/Applications/MAMP/htdocs/
├── index.php           # 前端页面
├── config.php          # 数据库配置文件
├── api/
│   ├── index.php       # API 入口（增删改查）
│   ├── db.php          # 数据库连接与初始化
│   ├── upload.php      # 文件上传接口
│   └── init.php        # 数据库初始化脚本（含初始数据）
├── admin/
│   └── index.php       # 后台管理页面
├── uploads/            # 上传文件存储目录
└── data/
    └── backups/        # 数据库备份目录
```
